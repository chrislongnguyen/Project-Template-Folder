# PEOPLE

**Purpose:** Team member profiles, role descriptions, and RACI assignments for this project. Maps to the EU (Effective Users) component in the 8-component system model.

**What goes here:**
- One file per person: `{first-last}.md`
- Role description, responsibilities, and RACI column for each workstream
- Onboarding notes and decision authority

**RACI reminder:**
- R = Responsible (does the work)
- A = Accountable (signs off — one per decision)
- C = Consulted (provides input)
- I = Informed (kept in the loop)

**Downstream reference:** RACI rows in this directory are the authoritative source for `1-ALIGN/` stakeholder maps and `3-PLAN/` risk owner assignments.
