# KEY RESULTS
## Measurable Results with Formulas
### Derived From: Derived Truth #1 (3 Pillars); Wiki KPI Design Standards

---

## Key Principle
> KPIs without formulas are aspirations, not metrics.

Each Key Result must specify:
1. Which effectiveness **pillar** it measures (Sustainability / Efficiency / Scalability)
2. The system **objective** it tracks
3. The business **impact** it reflects
4. A clear **description**
5. An explicit **calculation formula**
