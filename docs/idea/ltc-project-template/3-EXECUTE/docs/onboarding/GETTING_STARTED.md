## Welcome to This Project

### Step 1: Understand WHY (5 min)
Read `1-ALIGN/charter/PROJECT_CHARTER.md` to understand:
- Why this project exists
- Who it serves
- What success looks like

### Step 2: Understand WHAT CAN GO WRONG (10 min)
Read `2-PLAN/risks/UBS_REGISTER.md` to understand:
- The top risks this project faces
- How they are being mitigated
- What to watch out for

### Step 3: Understand HOW (15 min)
Read `2-PLAN/architecture/SYSTEM_DESIGN.md` to understand:
- The 7-component system design
- Technology choices and why they were made
- Key ADRs in `2-PLAN/architecture/ADRs/`

### Step 4: Set Up Your Environment (varies)
Follow the setup instructions in:
- `3-EXECUTE/config/` for environment configuration
- `3-EXECUTE/config/security/SECURITY_POLICY.md` for security requirements

### Step 5: Understand the Agent Configuration
If using Cursor / Antigravity / Claude Code:
- `CLAUDE.md` — Claude Code behavioral rules
- `AGENTS.md` — Antigravity agent config
- `.mcp/README.md` — MCP server integrations

### Step 6: Start Contributing
- Pick a task from `2-PLAN/roadmap/EXECUTION_PLAN.md`
- Follow the quality gates in `3-EXECUTE/tests/quality-gates/`
- Document decisions in `1-ALIGN/decisions/`
