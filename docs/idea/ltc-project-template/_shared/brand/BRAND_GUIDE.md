## 60-30-10 COLOR RULE
- **60%** primary (Midnight Green or White depending on background)
- **30%** secondary (Gold or Midnight Green)
- **10%** accent (Ruby Red or other accents)

## TYPOGRAPHY
- **Primary Font:** Tenorite (fallback: Calibri → Segoe UI → sans-serif)
- **Headlines:** 50pt / H1
- **Sub-Title:** 20pt / H2
- **Body Title:** 16pt / H3
- **Body Text:** 12pt / paragraph

## TONE
- Professional, analytical, direct
- Evidence-based and precise
- Structured outputs with clear visual hierarchy
- Cite LTC frameworks explicitly
