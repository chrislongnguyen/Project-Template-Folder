# LTC TYPOGRAPHY STANDARDS

## Primary Typeface
**Tenorite** — Primary body font for all documents and presentations.

### Fallback Chain
1. Tenorite (preferred)
2. Calibri
3. Segoe UI
4. Clean sans-serif system font

## Type Hierarchy

| Level | Size (Presentations) | Size (Documents) | Use |
|-------|---------------------|-------------------|-----|
| Headlines | 50pt | 24pt | H1 / Page titles |
| Sub-Title | 20pt | 18pt | H2 / Section headers |
| Body Title | 16pt | 14pt | H3 / Subsection headers |
| Body Text | 12pt | 11pt | Paragraphs / content |

Scale proportionally for different media (~40% reduction for documents vs. slides).
