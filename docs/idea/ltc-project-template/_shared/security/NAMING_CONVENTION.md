# LTC NAMING CONVENTION
### Derived From: Wiki — Naming System, Data Naming Rules

---

## Standard Format
```
[CLASSIFICATION]_[GROUP OWNER]_[MEMBER OWNER]_PROJECT ID. PROJECT NAME - ITEM ID. Item Name
```

## Rules
- Only prefix CLASSIFICATION if RESTRICTED, CONFIDENTIAL, or EXTERNAL
- Omit classification prefix for INTERNAL and PERSONAL
- Use underscores `_` to separate major segments
- Use dots `.` to separate IDs from names
- Use hyphens `-` within names for readability

## Examples
```
CONFIDENTIAL_INV TECH_John_P01. Trading Platform - D01. System Architecture
[LTC ALL]_WIKI - 1. DRAFT Wiki to LTC's Systems
[EXT]_CORP DEV_Jane_P02. Client Report - R01. Q1 Performance
```
