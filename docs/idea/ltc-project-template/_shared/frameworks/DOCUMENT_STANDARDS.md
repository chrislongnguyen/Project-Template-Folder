## 4 Truths of Effective Documents
1. **Reduce cognitive load** — The reader should not have to work hard to understand
2. **Establish credibility** — Evidence-based, well-structured, professionally presented
3. **Convey clear message** — One main point per section, no ambiguity
4. **Ensure maintainability** — Easy to update, version-controlled, living documents

## 5 Design Principles
1. **Clear hierarchy** — Use headings, subheadings, and visual grouping consistently
2. **60-30-10 color rule** — Primary 60%, Secondary 30%, Accent 10%
3. **Consistent patterns** — Same formatting for same types of content throughout
4. **Grouped proximity** — Related items are visually close; unrelated items are separated
5. **Negative space** — White space is not wasted space; it aids readability
