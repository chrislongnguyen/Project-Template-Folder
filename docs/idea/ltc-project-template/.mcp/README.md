## LTC Approved Integrations
Per LTC's EOE framework, the following integrations are approved:
- **ClickUp** — Project management, task tracking, OKR monitoring
- **Google Workspace** — Email, Drive, Calendar
- **GitHub** — Version control, CI/CD
- **Proton Suite** — Secure email, password management, VPN

## Adding a New Server
1. Document the server's purpose and which LTC system it connects to
2. Add the configuration to `servers.json`
3. Create an ADR in `1-ALIGN/decisions/` explaining why this server is needed
4. Update this README
