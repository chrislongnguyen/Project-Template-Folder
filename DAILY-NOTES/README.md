# DAILY-NOTES

**Purpose:** Daily notes, reflections, and captured observations for your project. Used as the primary input source for your Obsidian vault's daily-note workflow.

**What goes here:**
- Daily standup notes
- Meeting observations and quick decisions
- Captured blockers, ideas, and questions awaiting triage

**Naming convention:** `YYYY-MM-DD.md` (one file per day)

**Downstream routing:** Triage daily notes into the appropriate workstream artifact weekly. Unresolved items after 2 weeks belong in `inbox/`.
