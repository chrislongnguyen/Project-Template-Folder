# Simple Skill Template

Use for skills with 40 or fewer body lines. Copy and fill in the placeholders.

```markdown
---
name: {skill-name}
description: "{What this does and when. Use action verbs. Be specific about triggers.}"
---

# {Skill Title}

{1-2 sentence purpose.}

## When to Use

- {Trigger condition 1}
- {Trigger condition 2}

## Steps

1. {Step}
2. {Step}

## Constraints

- {Rule 1}
- {Rule 2}
```

## Checklist

- [ ] `name` is kebab-case and describes the capability
- [ ] `description` is 50+ characters with trigger language ("Use when...", "Create X when...")
- [ ] Steps are concrete actions, not vague instructions
- [ ] Separate `gotchas.md` file exists (even for simple skills)
