# Application Profiles — LTC Brand

Condensed from `rules/brand-identity.md`. If anything disagrees, the rules file wins.

## Web (HTML, CSS, SVG, React, canvas)

- Embed fonts: Google Fonts links for Inter and Work Sans (see palette-and-type.md).
- Light page: text often Dark Gunmetal on White; headers/accents Midnight Green and Gold.
- Dark page: Gold for emphasis text where appropriate; Midnight Green surfaces as designed.
- Charts: map series to LTC palette; avoid Chart.js/defaults blue/orange unless they match LTC hex values.

## Print-like (PDF export, static one-pagers)

- Same colors and type hierarchy; ensure embedded fonts or outline text if the PDF pipeline allows.
- Base body **11pt** relative scale as in the rules table.

## MS Office–aligned (Excel, Word, PPT themes)

- Use the **MS Office Theme** slots from `rules/brand-identity.md` (Accent 1–6 order) when setting theme colors in tools that mirror that model.
