# Palette and Typography — Quick Reference

**Canonical source:** `rules/brand-identity.md` at repo root.

## Must-not-miss primaries

| Role | Name | Hex |
|------|------|-----|
| Dark primary | LTC Midnight Green | `#004851` |
| Accent primary | LTC Gold | `#F2C75C` |
| Dark neutrals | LTC Dark Gunmetal | `#1D1F2A` |
| Light neutrals | LTC White | `#FFFFFF` |

## Typography

| Locale | Family | Google Fonts |
|--------|--------|--------------|
| English | Inter | `family=Inter:wght@400;500;600;700` |
| Vietnamese | Work Sans | `family=Work+Sans:wght@400;500;600;700` |

- Base size **A = 11pt**; scale headlines per rules table in `brand-identity.md`.
- CSS stacks may list `system-ui` as fallback **after** named fonts — never omit the Google Fonts include.

## Logo wording

- Company name in logo contexts: **LT Capital Partners**
- Midnight Green mark on light backgrounds; Gold mark on dark backgrounds (hex above).
