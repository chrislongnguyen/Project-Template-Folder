# Effective Thinking — Critical Thinking & Cognitive Biases

## The Critical Thinking Model

Every analysis follows this structure:

```
HYPOTHESIS  +  PREMISES  +  REASONING  →  CONCLUSION
```

### 1. Hypothesis (The Problem Statement)

Use the Situation-Complication framework:

- **Situation**: Who does what to whom, when, where, how, why?
- **Complication**: What breaks, fails, or is unknown?

### 2. Premises (The Evidence)

Three types of premises, in order of reliability:

| Type | Definition | Reliability |
|---|---|---|
| **Truths** | Verified, tested, proven | Highest |
| **Assumptions** | Believed but not verified (each = a risk if wrong) | Medium |
| **Facts** | Observed but not yet explained | Varies |

### 3. Reasoning Methods

In order of preference:

| Method | How It Works | When to Use |
|---|---|---|
| **Deduction** | General truth -> specific conclusion | PREFERRED — most reliable |
| **Abduction** | Observations -> best explanation | ACCEPTABLE — when deduction isn't possible |
| **Induction** | Observations -> generalization | USE WITH CAUTION — weakest form |

### 4. Conclusion

Must follow logically from premises + reasoning. If the conclusion feels wrong, challenge the premises before challenging the conclusion.

## The Five Cognitive Biases (UBS of Human Thinking)

These are the primary failure modes of human reasoning. Check for them explicitly.

| Bias | What It Does | Defense |
|---|---|---|
| **Availability** | Over-weight recent/vivid information | Seek base rates and historical data |
| **Representativeness** | Assume patterns that don't exist | Check sample size and statistical significance |
| **Anchoring** | Over-weight the first data point | Generate multiple independent estimates |
| **Affect** | Let emotions drive conclusions | Separate data gathering from evaluation |
| **Confirmation** | Seek evidence that confirms existing beliefs | Actively seek disconfirming evidence |

## Using This in Practice

- Use the **Spike Investigation** template for time-boxed investigations
- Always include a Bias Check section in important analyses
- When disagreeing with someone, challenge premises before conclusions
