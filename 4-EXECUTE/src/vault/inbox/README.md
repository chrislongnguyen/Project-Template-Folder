# inbox/

Staging area for all agent-created notes awaiting human review.

All multi-note agent writes land here first (AP3). Human reviews and promotes
notes to the appropriate folder (`projects/`, `research/`, etc.).

Agents must not move or rename files out of inbox — promotion is a human action.
Maximum 1 new note per agent session unless human grants a batch exception in-session.

Notes are accessible via CLI by default (AP4 opt-out model).
